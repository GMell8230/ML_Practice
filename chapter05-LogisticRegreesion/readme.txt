# Logistic回归-最优化算法
logistic思想：根据现有数据对分类边界线建立回归公式，以此进行分类。
“回归”一词源于最佳拟合，表示要找到的最佳拟合参数集

logistic回归
优点：计算代价不高，易于理解和实现
缺点：容易欠拟合，分类精度可能不高
使用数据类型：数值，标称

sigmoid函数 0，1
实现Logistic回归分类器，可以在每个特征上都乘以一个回归系数，然后将所有的
结果值相加，将这个总和带入sigmoid函数之中，进而得到一个范围在0~1之间的数值
大于0.5的数据分入1类，小于0.5分入0类，Logistic回归看作是一个概率估计。

回归系数的确定：
z = w0*x0 + w1*x1 +...+ wn*xn
z为sigmoid函数输入，x是分类器的输入数据，向量w是要找到的最佳参数
