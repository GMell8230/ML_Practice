# 基于概率论的分类方法：朴素贝叶斯

优点：在数据较少的情况下仍然有效，可以处理多类别问题
缺点：对于输入数据的准备方式较为敏感
适用数据类型：标称型 即目标变量的结果要在有限的目标集之中

朴素贝叶斯的一般过程
1）收集数据:[例如：rss(Really Simple Syndication)源等]
2）准备数据：数值型或布尔型数据
3）分析数据：特征量较多时，直方图绘制效果更明显
4）训练算法：计算不同的独立特征的条件概率
5）测试算法：计算错误率
6）使用算法：一个常见的贝叶斯应用是文档分类。可以在任意的场景使用贝叶斯分类器

文档分类的两个假设：
1）特征之间相互独立，即一个单词出现的可能性与前后无关（朴素（naive））；#由此可以降低所需的样本数
2）每个特征同等重要

nativeBayes.py 对于字符列表进行词向量的预处理，然后计算每个词在语句出现的概率，进而实现朴素贝叶斯的分类
emailFilter.py 通过朴素贝叶斯实现垃圾邮件的过滤
